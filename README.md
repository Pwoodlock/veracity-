# Veracity

**Version:** 0.0.1-alpha

> ⚠️ **ALPHA SOFTWARE** - This project is in early development. Features and APIs may change.

## Tech Stack

### Backend
- **Ruby** 3.3.5
- **Rails** 8.0
- **PostgreSQL** 14+
- **Redis** 7+
- **Sidekiq** - Background job processing

### Infrastructure
- **SaltStack** - Configuration management & orchestration
  - Salt Master
  - Salt API
- **Caddy** v2 - Reverse proxy with automatic HTTPS

### Frontend
- **DaisyUI** - Tailwind CSS component library
- **ApexCharts** - Data visualization
- **Node.js** 18 LTS

### Monitoring & Integrations
- **Python** 3.x (venv)
- **PyVulnerabilityLookup** - CVE scanning
- **Gotify** - Push notifications (Docker)
- **Proxmox API** - VM management
- **Hetzner Cloud API** - Cloud infrastructure

### Deployment
- **Puma** - Application server
- **Systemd** - Service management
- **UFW** - Firewall
