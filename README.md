## Overview

Veracity is a powerful server infrastructure management platform built on SaltStack, designed to provide **infrastructure truth** through intelligent automation and real-time monitoring. With its distinctive SALT Crystal V branding, Veracity combines the robust capabilities of SaltStack with a modern Ruby on Rails interface to deliver seamless server management at scale.

### Key Features

- **SALT-Powered Automation** - Leverages SaltStack for reliable, scalable infrastructure management
- **CVE Vulnerability Monitoring** - Automated CVE scanning with PyVulnerabilityLookup integration, Gotify notifications, and CISA KEV tracking
- **Real-Time Monitoring** - Live server status, metrics, and health checks
- **Automated Security Updates** - Scheduled patching and compliance management
- **Backup Management** - Automated backup scheduling and verification
- **Group Management** - Organize servers into logical groups with bulk operations
- **Command Execution** - Run commands across multiple servers simultaneously
- **Metrics & Analytics** - Historical performance data and trend analysis
- **Role-Based Access Control** - Secure multi-user authentication with Devise
- **Beautiful Dark UI** - SALT Crystal V theme with cyan accents and Inter typography
